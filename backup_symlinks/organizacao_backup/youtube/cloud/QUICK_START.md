## 💡 DICAS PRO

1. **Primeira vez:** Pode demorar 2-3 min para instalar dependências
2. **Sessão expirou:** Reconecte e reexecute (sempre baixe antes!)
3. **Sem GPU:** Runtime > Change runtime type > GPU
4. **Vídeo muito grande:** Reduza bitrate no código (5000k → 3000k)

---

## 🆘 PROBLEMAS COMUNS

### "Runtime desconectou"
- Normal após 12h ou inatividade
- Reconecte e execute novamente
- **Sempre baixe arquivos antes de fechar!**

### "Erro ao renderizar"
- Verifique se áudio é MP3/WAV válido
- Tente preset='fast' ao invés de 'medium'
- Reduza duração do vídeo

### "Download não inicia"
- Navegador pode bloquear múltiplos downloads
- Permita downloads do Colab nas configurações
- Ou baixe manualmente da pasta lateral (📁)
